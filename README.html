Generating Maximal Partial Room Squares in R

Matthew Henderson

  • I: greedy1
  • II: greedy2
  • References

Two different greedy procedures for building maximal partial Room squares in R inspired by Meszka and Rosa (2021).

I: greedy1

greedy1 visits all cells in order placing into the next cell the first available pair not violating the conditions of being a partial Room square.

library(tictoc)

n <- 10

tic()
R <- greedy1(n)
toc()
#> 0.172 sec elapsed

# is R a maximal partial Room square?
is_maximal_proom(R)
#> [1] TRUE

II: greedy2

greedy2 iterates through all pairs in order placing the next pair in the first available cell not violating the conditions of being a partial Room square.

n <- 10

tic()
R <- greedy2(n)
toc()
#> 0.232 sec elapsed

# is R a maximal partial Room square?
is_maximal_proom(R)
#> [1] TRUE

References

Meszka, Mariusz, and Alexander Rosa. 2021. Maximal Partial Room Squares. Journal of Combinatorial Designs 29 (7): 482501. https://doi.org/10.1002/jcd.21777.
